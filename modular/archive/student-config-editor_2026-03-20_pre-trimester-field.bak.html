How to customize your student's experience

Each tab controls a different part of the app:

🎛️ Modes — Rename modes (e.g., "Bad Brain Day" → "Concussion Day")

🥄 Spoon Plan — Set daily tasks, spoon costs, and energy budget

🍽️ Meals — Customize easy meal suggestions for recovery mode

📚 Quizzes — Add custom study quizzes + hide built-in ones

💜 Comfort — Customize scriptures, support messages, breathing timing

⚠️ Alerts — Set when the app warns about spoon debt

✨ AI Assistant — Use your own AI to generate config (paste prompt, get JSON back)

💡 Tip: Changes won't affect other students. If you leave a section empty, the app uses smart defaults.

💡 Tip: Use the Export button to save a backup before making big changes.
